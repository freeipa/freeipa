% Running header/footer showing the current section (or "Appendix:
% <section>" once past the appendix divider -- see
% TeX/appendix-divider.md, which redefines \sectionmark again) and
% page number, so a reader can always tell where in the document they
% are without the table of contents. Section goes in the left header,
% subsection (when one is active) in the right header -- the classic
% two-level running-header split, chosen over concatenating both into
% one side after that produced headers so long they visually collided
% with the (now-removed) static "ipacta Documentation" left-hand
% label, e.g. on the very first section, "ipacta -- Comprehensive
% Documentation -- Process Isolation and Security Model" ran together
% with it. \sectionmark leaves the right mark empty so a section's own
% first page (no subsection yet) shows nothing on the right.
